# Changelog

## 1.0.0 (2026-03-29)

Initial public release.

- Drag-and-drop or file picker to load folders of markdown files
- Dark "Ink & Paper" theme with teal accents
- Sidebar navigation tree with nested folder support
- Syntax-highlighted code blocks (30+ languages via Shiki)
- Zoom controls (1x, 1.25x, 1.5x presets + slider)
- Scroll position memory per document
- Auto-reload when files change on disk (2s polling)
- IndexedDB persistence across page reloads
- Add/remove/reload folders independently
- Truncated name overlay on hover
- File System Access API with fallback for older browsers
- Fully client-side — no backend, no data leaves your machine
- Docker deployment support
